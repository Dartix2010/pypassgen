# PyPassGen

**A simple, secure and fast command-line password generator written in Python.**

Perfect for developers, sysadmins, and anyone who needs strong passwords in seconds.

## ✨ Features
- Customizable length (default: 12 characters)
- Options to include/exclude uppercase letters, numbers, and symbols
- Generate 1 or multiple passwords at once
- 100% Python standard library (no external dependencies)
- Easy to use with `--help`
